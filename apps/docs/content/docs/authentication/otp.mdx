## Usage

<Callout type="info" title="Note">
  This block uses `react-hook-form`, `zod`, and `@hookform/resolvers`
</Callout>

<Callout type="info" title="Auto-fill Support">
  The OTP component supports auto-fill on iOS with `textContentType="oneTimeCode"` and on Android with
  `autoComplete="sms-otp"` for seamless SMS code verification.
</Callout>
